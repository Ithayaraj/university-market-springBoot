# UniMarket Frontend - React + Vite

A premium, high-performance frontend for the University Market Place, built with React, Vite, and modern CSS.

## Technology Stack
- **React 18**: Core UI library.
- **Vite**: Ultra-fast build tool and dev server.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **Framer Motion**: Powerful animations and transitions.
- **Lucide React**: Clean and consistent icon set.
- **Axios**: Promised-based HTTP client for API interactions.

## Features
- **Modern UI/UX**: Premium design with glassmorphism, smooth animations, and dark mode support.
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewports.
- **Dynamic Routing**: Client-side navigation with React Router.
- **Context API**: Global state management for Authentication and Notifications.
- **Profile Customization**: Advanced profile editor with image upload and persistence.
- **Product Marketplace**: Interactive product browsing and management.
- **Messaging System**: Real-time chat interface for buyer-seller communication.

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`

## Project Architecture
- `/src/components`: Reusable UI components.
- `/src/pages`: Main application views.
- `/src/context`: State management providers.
- `/src/api`: Axios instance and interceptors.
- `/src/assets`: Static assets and global styles.

## Environment Variables
Ensure the backend API URL is correctly configured in `src/api/axios.js`.
